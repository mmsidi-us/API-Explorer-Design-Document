# REST API Explorer & Design Document

This repository contains a two-part project demonstrating both the consumption and architectural design of RESTful APIs. Part 1 focuses on building a script to interact with real-world public endpoints, while Part 2 details an enterprise-grade API design document for a custom culinary Recipe Manager platform.

## 📁 Repository Structure

```text
REST API Explorer & Design Document/
├── README.md                # Project overview and execution instructions
├── requirements.txt         # Project Python dependencies
├── api_explorer.py          # Runnable Python script consuming 3 public APIs
├── api_documentation.md     # Engineering log of public APIs explored
├── api_design.md            # Comprehensive REST design document (Recipe Manager)
└── curl_examples.txt        # Language-agnostic curl commands for rapid testing
